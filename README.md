## 📝 À propos du projet

**ApproLogApp** est une application web robuste conçue pour simplifier la gestion quotidienne d'une entreprise. Elle intègre plusieurs modules critiques pour le suivi administratif et opérationnel.

### Fonctionnalités clés :
- **Gestion Administrative** : Création de devis, facturation PDF et notes de livraison.
- **Gestion de Stock** : Suivi des produits, des consommables et des achats.
- **Ressources Humaines** : Gestion des employés, bulletins de salaire et suivi des congés.
- **Suivi Financier** : Livre de caisse, suivi des taxes, factures d'eau/électricité et loyers.
- **Maintenance Industrielle** : Suivi de la rentabilité et de l'usure des machines.

## 🛠️ Stack Technique

- **Backend** : PHP 8.1+ / Laravel 10
- **Frontend** : Blade, Bootstrap, JavaScript (Vite)
- **Base de données** : MySQL
- **Reporting** : Chart.js pour les graphiques de rentabilité
